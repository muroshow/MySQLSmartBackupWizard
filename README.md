# 🧙‍♂️ MySQL Smart Backup Wizard (Cross-Platform)

## 🇹🇷 Türkçe Versiyon

Bu araç, macOS ve Windows sistemlerinde MySQL veritabanlarınızı ve kullanıcı yetkilerinizi saniyeler içinde yedeklemeniz için tasarlanmış akıllı bir sihirbazdır.

### ✨ Öne Çıkan Özellikler

* **🌍 Otomatik Dil:** Sistem dilinize göre Türkçe veya İngilizce arayüz.
* **📦 ZIP Sıkıştırma:** Yedeklerinizi ZIP formatında sıkıştırarak disk alanı tasarrufu sağlar.
* **💻 Çapraz Platform:** Hem macOS hem Windows desteği.

### 🚀 Hızlı Kurulum

**🍎 macOS:**

```bash
curl -L -o ~/Desktop/"MacOS MySQL Smart Backup Wizard.command" "https://raw.githubusercontent.com/muroshow/MySQLBackup/main/MacOS%20MySQL%20Smart%20Backup%20Wizard.command" && chmod +x ~/Desktop/"MacOS MySQL Smart Backup Wizard.command"

```

**🪟 Windows:**

```powershell
powershell -Command "iwr -outf $HOME\Desktop\'Windows MySQL Smart Backup Wizard.ps1' 'https://raw.githubusercontent.com/muroshow/MySQLBackup/main/Windows%20MySQL%20Smart%20Backup%20Wizard.ps1'"

```

## 🇺🇸 English Version

A smart, interactive CLI tool designed to backup your MySQL databases and user privileges effortlessly on both macOS and Windows.

### ✨ Key Features

* **🌍 Auto-Language:** Automatically switches between English and Turkish.
* **📦 ZIP Compression:** Save disk space by compressing backups.
* **💻 Cross-Platform:** Supports both macOS and Windows.
